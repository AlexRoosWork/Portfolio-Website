Doom Emacs

2021-03-02 :: [Alex Roos]
#opensource  #emacs 

IDEs I have used⌗

When I started to code with Python, my first IDE was PyCharm. In contrast to the clunky Java IDE I had tried to install early on when I tried to .#learn2code, PyCharm was easy to use and looked good, too!

Nevertheless, I then switched to VSCodium (basically VSCode, minus the telemetry), because of recommendations from friends.

Doom⌗

But then, in March of 2020, I was recommended Doom Emacs. This lead me down the Emacs rabbit hole: learning all the different key chords, tools, workflows and even a little elisp.

Doom takes a central place in my personal setup now and I use it for all kinds of purposes:

  • org-mode to organise thoughts and write blogs such as this.
  • tramp to ssh into remote machines and edit files within Doom.
  • magit to interact with git repos
  • dired to manage files
  • the different language modes as my IDE (mostly Python, JavaScript and HTML)
